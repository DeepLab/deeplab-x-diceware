### DeepLab x Diceware

Simply, `python dicewarez.py`.

#### Custom wordlists?

Yep! Add a file called `custom.wordlist.txt` here, and those words will be added into the mix.  For example:

```
vacmaster
shana
bbs
old-skool
hamburguesa
netflix-and-chill
bae
basic-bitch
facebook
hotline-bling
facetime
trap-queen
raspberry-pi
desktopmancie
bombaclot
showverdose
on-fleek
```
